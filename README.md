# Trabalho – Estruturas de Dados

Este trabalho consiste na implementação e análise de desempenho de algoritmos de ordenação, com foco no QuickSort com partição em três vias (3-way) e no Merge Sort Externo. O objetivo é comparar o tempo de execução dos algoritmos em diferentes conjuntos de dados, incluindo cenários com grandes volumes de informações.

## Tecnologias
- Python 3

## Algoritmos
- QuickSort com partição em três vias
- Merge Sort Externo utilizando arquivos e intercalação de blocos

## Resultados
Foram realizados testes de tempo de execução e gerados gráficos para análise comparativa do desempenho dos algoritmos. Os resultados e discussões estão apresentados no relatório do trabalho
